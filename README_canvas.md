# Business Model Canvas — Arquivos do Projeto

## Arquivos

| Arquivo | Descrição |
|---------|-----------|
| `business-model-canvas.html` | **Versão standalone** — abre direto no navegador com duplo clique. Toda a funcionalidade do canvas, templates, edição e exportação funcionam offline. Persiste dados no localStorage do navegador. |
| `business-model-canvas.jsx` | **Versão React (artifact)** — funciona dentro do Claude (claude.ai) com o painel de IA integrado. |

## Como usar

1. Baixe os dois arquivos
2. Coloque em: `F:\Projeto de impressoras\SACKS CMA\referencias\canvas_preincubados\`
3. Para usar no dia a dia: abra o `.html` com duplo clique no Explorer
4. Para usar com IA: abra o `.jsx` dentro de uma conversa no Claude

## Nota sobre o painel de IA (GRÁTIS)

A IA usa o **Google Gemini 2.0 Flash** — totalmente gratuito. Para ativar:

1. Acesse [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
2. Faça login com sua conta Google
3. Clique em "Create API Key" — a chave começa com `AIzaSy...`
4. Abra o `business-model-canvas.html` no navegador
5. No painel "Estrategista IA", cole a chave no campo 🔑
6. Pronto! A chave fica salva no navegador, só precisa inserir uma vez

O Gemini Flash é gratuito com limite de 15 requisições por minuto / 1.500 por dia — mais que suficiente para uso normal.
